# Code Explainer Platform

An AI-powered developer learning platform that analyzes source code and generates step-by-step explanations, complexity metrics, flowcharts, and refactoring insights.

The platform helps beginners and developers understand how code works internally and how it can be improved.

---

## Live Demo

https://code-explainer-platform.vercel.app/

---

## Key Features

### Code Analysis

* Detects functions, loops, decisions, and nesting depth
* Calculates cyclomatic complexity
* Provides maintainability score

### Line-by-Line Explanation

* Explains each line of code
* Describes what the line does, why it exists, and its impact

### Flowchart Generation

* Automatically generates visual program flow
* Helps visualize execution paths

### AI Tutor

* Ask questions about the analyzed code
* Get step-by-step explanations and debugging guidance

### Code Smell Detection

* Identifies complexity hotspots
* Highlights maintainability risks

### Refactoring Suggestions

* Suggests improvements for cleaner architecture
* Recommends function extraction and modularization

### Code Metrics Dashboard

Displays:

* Total lines
* Functions
* Loops
* Decision points
* Cyclomatic complexity
* Maintainability score
* Time complexity estimation

## How It Works

1. User submits code.
2. Backend parses structure and extracts code metrics.
3. AI model generates explanations and insights.
4. System visualizes flowcharts and analysis reports.
